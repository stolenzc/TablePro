# Phím tắt

TablePro hỗ trợ phím tắt phong phú để giúp bạn làm việc nhanh hơn mà không cần dùng chuột. Học các phím tắt này để tăng tốc đáng kể quy trình làm việc database của bạn.

## Tham Khảo Nhanh

### Phím Tắt Thiết Yếu

| Hành động | Phím tắt |
|--------|----------|
| Thực thi query | `Cmd+Enter` |
| Kết nối mới | `Cmd+N` |
| Mở lịch sử | `Cmd+Y` |
| Cài đặt | `Cmd+,` |
| Đóng cửa sổ | `Cmd+W` |
| Thoát | `Cmd+Q` |

## Trình Soạn Thảo SQL

### Thực Thi Query

| Hành động | Phím tắt | Mô tả |
|--------|----------|-------------|
| Thực thi query | `Cmd+Enter` | Chạy query tại con trỏ, hoặc tất cả câu lệnh đã chọn tuần tự |
| Giải thích query | `Option+Cmd+E` | Hiển thị kế hoạch thực thi cho query tại con trỏ |
| Định dạng SQL | `Option+Cmd+F` | Định dạng query SQL (làm đẹp và chuẩn hóa cú pháp SQL) |

<Tip>
`Cmd+Enter` là phím tắt quan trọng nhất. Nó chạy query mà con trỏ của bạn đang ở trong đó, hoặc thực thi văn bản đã chọn. Nếu phần chọn chứa nhiều câu lệnh, chúng sẽ được thực thi tuần tự trong một transaction.
</Tip>

## Phím Tắt Chế Độ Vim

Khi chế độ Vim được bật (**Settings** > **Editor** > **Editing** > **Vim mode**), trình soạn thảo SQL sử dụng phím tắt theo chế độ. Huy hiệu chỉ báo chế độ xuất hiện trên thanh công cụ.

### Các Chế Độ

| Phím | Hành động |
|------|-----------|
| `Escape` | Quay lại chế độ Normal |
| `i` / `I` | Chèn trước con trỏ / ở đầu dòng |
| `a` / `A` | Chèn sau con trỏ / ở cuối dòng |
| `o` / `O` | Mở dòng mới bên dưới / bên trên |
| `v` | Chế độ Visual (theo ký tự) |
| `V` | Chế độ Visual Line (theo dòng) |
| `:` | Chế độ Command-line |

### Điều Hướng (Chế Độ Normal)

| Phím | Hành động |
|------|-----------|
| `h` / `j` / `k` / `l` | Trái / Xuống / Lên / Phải |
| `w` / `b` / `e` | Từ tiếp theo / Từ trước / Cuối từ |
| `0` / `$` | Đầu dòng / Cuối dòng |
| `^` / `_` | Ký tự không trống đầu tiên của dòng |
| `gg` / `G` | Dòng đầu tiên / Dòng cuối cùng |

### Toán Tử (Chế Độ Normal)

| Phím | Hành động |
|------|-----------|
| `dd` | Xóa dòng |
| `yy` | Yank (sao chép) dòng |
| `cc` | Thay đổi dòng |
| `x` | Xóa ký tự |
| `p` / `P` | Dán sau / trước con trỏ |
| `d` + di chuyển | Xóa với di chuyển |
| `y` + di chuyển | Yank với di chuyển |
| `c` + di chuyển | Thay đổi với di chuyển |

### Chế Độ Visual

| Phím | Hành động |
|------|-----------|
| `d` | Xóa vùng chọn |
| `y` | Yank vùng chọn |
| `c` | Thay đổi vùng chọn |

### Chế Độ Command-line

| Lệnh | Hành động |
|-------|-----------|
| `:w` | Thực thi truy vấn (`Cmd+Enter`) |
| `:q` | Đóng tab |

### Tiền Tố Số Đếm

Thêm số trước các phím di chuyển và toán tử: `3j` (xuống 3 dòng), `2dd` (xóa 2 dòng), `5x` (xóa 5 ký tự).

<Note>
Phím tắt chế độ Vim chỉ hoạt động trong trình soạn thảo SQL và không ảnh hưởng đến bảng dữ liệu hoặc các panel khác. Các phím tắt chuẩn như `Cmd+Enter` vẫn có sẵn trong tất cả các chế độ.
</Note>

### Gán lại Phím tắt

1. Mở **Settings** (`Cmd+,`) và chọn tab **Keyboard**
2. Tìm hành động bạn muốn gán lại (sử dụng ô tìm kiếm để lọc)
3. Click vào ô phím tắt bên cạnh hành động
4. Nhấn tổ hợp phím mới bạn muốn gán
5. Phím tắt được cập nhật ngay lập tức trong thanh menu

### Xóa Phím tắt

Để xóa hoàn toàn một phím tắt:

1. Click vào ô phím tắt của hành động
2. Nhấn `Delete` hoặc `Backspace`
3. Phím tắt được xóa — mục menu sẽ không có phím tắt

### Phát hiện Xung đột

Khi bạn gán một phím tắt đã được sử dụng bởi hành động khác, TablePro hiển thị hộp thoại xác nhận:

- **Cancel** — giữ nguyên phím tắt hiện tại
- **Reassign** — chuyển phím tắt sang hành động mới (phím tắt của hành động trước đó bị xóa)

<Warning>
Các phím tắt hệ thống (`Cmd+Q`, `Cmd+H`, `Cmd+M`, `Cmd+,`) không thể gán lại. TablePro sẽ hiển thị cảnh báo nếu bạn cố sử dụng chúng.
</Warning>

### Đặt lại về Mặc định

Nhấp **Reset to Defaults** ở cuối cài đặt Keyboard để khôi phục tất cả phím tắt về giá trị ban đầu.

### Danh mục Phím tắt

Phím tắt được tổ chức theo danh mục:

| Danh mục | Ví dụ |
|----------|-------|
| **File** | New Connection, New Query Tab, Close Tab, Save Changes |
| **Edit** | Copy as CSV, Copy as JSON, Export, Import |
| **View** | Toggle Sidebar, Toggle Cell Inspector, Query History |
| **Tabs** | Next Tab, Previous Tab, New Table |
| **AI** | Toggle AI Chat, Explain with AI, Optimize with AI |

<Note>
Phím tắt chuyển tab (`Cmd+1` đến `Cmd+9`) tuân theo quy ước chuẩn macOS và không thể tùy chỉnh.
</Note>

## Mẹo Để Hiệu Quả

### Phím Tắt Được Dùng Nhiều Nhất

Tập trung học những phím này trước:

1. `Cmd+Enter` - Thực thi query
2. `Cmd+N` - Kết nối mới
3. `Cmd+Y` - Lịch sử query
4. `Cmd+F` - Tìm
5. `Cmd+C` / `Cmd+V` - Sao chép/Dán

### Mẹo Quy Trình Làm Việc

**Lặp query nhanh**:
1. Viết query trong editor
2. `Cmd+Enter` để chạy
3. Thực hiện thay đổi
4. `Cmd+Enter` lại

**Khám phá dữ liệu**:
1. Click bảng trong sidebar
2. Xem dữ liệu trong grid
3. `Cmd+Enter` với query tùy chỉnh
4. `Cmd+Y` để tìm query trước đó

**Chỉnh sửa dữ liệu**:
1. Điều hướng bằng phím mũi tên
2. `Enter` để chỉnh sửa ô
3. Gõ giá trị mới
4. `Tab` để chuyển đến ô tiếp theo
5. `Cmd+S` để lưu tất cả thay đổi

### Kỹ Thuật Ghi Nhớ

**Nhóm theo ngữ cảnh**:
- Tất cả phím tắt `Cmd+` là hành động chính
- Thêm `Shift` cho biến thể (ví dụ: `Cmd+Z` hoàn tác → `Cmd+Shift+Z` làm lại)
- Thêm `Option` cho phương án thay thế

**Mẫu thường gặp**:
- `Cmd+[chữ cái]` - Hành động thông thường (N=mới, W=đóng, Q=thoát)
- `Cmd+Shift+[chữ cái]` - Biến thể của hành động thông thường
- Phím mũi tên - Điều hướng
- `Tab` / `Shift+Tab` - Tiếp theo/Trước

## Khả Năng Truy Cập

### Hỗ Trợ VoiceOver

TablePro hoạt động với macOS VoiceOver:

- Điều hướng bằng phím `Control+Option+Mũi tên`
- Tương tác bằng `Control+Option+Space`
- Tất cả phần tử UI đều có nhãn

### Điều Hướng Chỉ Bằng Bàn Phím

Bạn có thể sử dụng TablePro hoàn toàn không cần chuột:

1. `Tab` di chuyển giữa các phần UI chính
2. Phím mũi tên điều hướng trong các phần
3. `Enter` hoặc `Space` kích hoạt control
4. `Escape` đóng dialog/menu

## Khắc Phục Sự Cố

### Phím Tắt Không Hoạt Động

1. **Kiểm tra focus**: Đảm bảo panel đúng có focus
2. **Kiểm tra xung đột**: Ứng dụng khác có thể đang bắt phím tắt
3. **Kiểm tra modifier**: Đảm bảo bạn đang nhấn đúng phím modifier

### Xung Đột Toàn Hệ Thống

Nếu phím tắt không hoạt động, kiểm tra **System Settings** > **Keyboard** > **Keyboard Shortcuts** để tìm xung đột với:

- Mission Control
- Spotlight
- Input Sources
- App Shortcuts
